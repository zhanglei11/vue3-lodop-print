#### 安装教程

1. yarn install
2. yarn run dev / yarn run serve
3. yarn run build

#### 使用说明

无预览静默打印纯前端处理，

1. 绕过预览模式打印有水印，只有交钱才能解决
2. 先预览后打印无任何问题，可以放心使用
3. 此打印机支持五花八门功能实在太多，本人只会一点使用皮毛
